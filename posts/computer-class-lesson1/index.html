前言>

前言 #

计算机是人类智慧的结晶，计算机科学一向被认为是最掉头发的学科之一，我计划分几次简单地向大家科普一些计算机理论知识，这次先从计算机的历史开始。

分析机>

分析机 #

空想中的计算机>

空想中的计算机 #

严格上来说，分析机(analytical engine)在其设计者，英国发明家查尔斯·巴贝奇(Charles Babbage)的一生都未被制造出来，它就如同达芬奇的手稿，直到巴贝奇逝世多年后才被发掘出其中的重要意义。一直到1991年，才终于有人按照其设计制造出了他梦想中的分析机(并且真的能用！)

分析机1991年的制成同样也说明了一件事，那就是它以19世纪的科技水平完全是可以建成的，而巴贝奇未能做到的主要原因就是他没有充足的资金以购置浩如烟海的机械部件，如果分析机真的在当年建成而投入使用，人类社会恐怕又要进步100年。

虽然没能亲眼看到自己最重要的发明:分析机制成，但巴贝奇的一生绝对称不上失败。他在25岁时就成为英国皇家学会会员，还是剑桥大学的数学教授，更是多家协会的创始会员，在取得如此成就的同时，他还享受了80岁的高寿(在19世纪)。

先进与落后并存>

先进与落后并存 #

在时代限制下，巴贝奇设计的分析机是一台由蒸汽驱动的机械设备。这意味着它的尺寸会十分巨大，达到了30米长、10米宽；并且运算速度不会太快，它甚至无法在1s内进行一次加法运算，在这方面它完全无法与它的电子计算机后辈们相提并论。

但同时与早期的电子计算机相比，它也拥有自己的优势，它的内存经过估算达到了20kb多，这已经要比早期的电子计算机要大不少了；除此之外，分析机还拥有自己的程序。

要讲程序，首先不得不提的是巴贝奇的一位女性朋友：阿达·洛芙莱斯，作为知名诗人的女儿，伯爵的夫人，阿达最不缺的就是钱和闲，她倾力为分析机开发算法，首次提出了计算机中的循环和子程序概念，同时还通过自己的人脉为分析机做宣传，她也被后世尊为世界上第一个程序员。

ENIAC>

ENIAC #

开创先河>

开创先河 #

ENIAC全名是Electronic Numerical Integrator And Computer，翻译过来就是电子数字积分计算机。ENIAC在1942年开始研制，初衷是用来计算二战中的弹道数据，但待其研制完成，二战也已经结束了。

天才人物冯·诺依曼在此时出现了，在改进ENIAC的过程中，他也同时在进行另一台更先进的计算机的研发，这就是EDVAC(Electroic Discrete Variable Automatic Computer,电子离散变量自动计算机)，并最终将ENIAC也改造成为了类似的计算机。下面是一个标志性的改动:ENIAC最初计划使用接线板来设计和存储程序，但后来改用了打孔纸来承载这一工作，这点对于早期的计算机工程师们来说，减轻了不少负担，大大提高了工作效率。

在积累了丰富的工作经验后，冯·诺依曼提出了著名的冯诺依曼体系结构，将计算机分为五个基本部分:输入设备、存储器、运算器、控制器、输出设备，时至今日，现在的计算机仍然属于"冯诺依曼机"

人物介绍:冯诺依曼>

人物介绍:冯诺依曼 #

约翰·冯·诺依曼(John von Neumann,1903~1957)是一位真正的全才，在进行电子计算机的研制工作之余，还作为原子能专家参与曼哈顿计划。他的冯诺依曼体系结构影响之深远上文已经叙述。但令人惋惜的是，这位伟大的全才在54岁时便因癌症而英年早逝。

物理知识:ENIAC的运行原理>

物理知识:ENIAC的运行原理 #

ENIAC是一个模块化的计算机，主要的结构有20个累加器，3个函数表，1个高速乘法器以及一个除法与开平方器。除此之外还有时钟周期模块等控制模块。这些模块的基本元件是真空管和二极管。

真空管>

真空管 #

真空管也就是电子管，有三个电极，其中一个是控制电极，根据这一控制电极的接电情况，真空管会改变自身的导电性能，从而进行复杂的电子运算。在ENIAC之前的一些电子计算机(ENIAC是第一台通用计算机，但不是第一台电子计算机)大都采用继电器实现类似的效果，但是由于继电器采用机械结构，在体积控制、使用寿命、工作效率等方面都远远不如真空管。在60年代，真空管还会被更加优秀的晶体管取代。

虽然ENIAC一般被视为电子管计算机，但在ENIAC中仍然使用了一部分的继电器。

二极管>

二极管 #

二极管具有单向导通性，在电路中可以起到开关、限流、稳压等作用。

ENIAC中二极管的具体作用我没有查到资料，后面查到了补充。

总结 #
