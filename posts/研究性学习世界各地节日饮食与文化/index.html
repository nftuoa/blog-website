资料搜集-原则 #

  1. 选取某一区域性文化圈内的代表性国家，该国家的代表性节日
  2. 所选取节日必须要有特定的食物(类比中国端午节)，在本国范围内无较大偏差(小偏差：类比甜粽子和咸粽子，大偏差：种类发生变化）
  3. 与之相关的传统文化，神话故事等
  4. 对该食物之所以成为节日食物的原因进行分析(区域物产等)

文化圈（摘自网络）及其代表国家（自拟） #

  1. 汉字文化圈:中国
  2. 拉丁文化圈:英国(这里实际上是英格兰)
  3. 伊斯兰文化圈:可视为一个整体
  4. 印度文化圈:印度
  5. 斯拉夫文化圈:俄罗斯
    别问我咋选的，问就是瞎选的

中国-端午节 #

  1. 为什么不选择春节：中国各地的春节饮食习惯迥异，难以汇总分析
  2. 传统食物：粽子(甜粽、咸粽)
  3. 文化：屈原投江(注:汨罗江沿岸并不出产糯米，该典故很可能与该习俗形成无关)
  4. 成因：中国物产丰富，粽子所用材料在中国各个地区都有种植，其能够在全国范围内流传千古应该也有取材方便的原因

英国——Plough Monday(犁地星期一)(还没找到这东西中文名)(等我把英语订正完问Jeff这玩意叫啥) #

Plough Monday - Wikipedia,节日介绍

  1. 传统食物：Plough Pudding(一种布丁)
  2. 文化:是1月6日之后的第一个星期一，也是圣诞节十二天后一切恢复正常的日子，人们将重返工作岗位。(这句是复制来的)也是新的一年耕作的开始
  3. 发源于诺福克郡
  4. 原料：面粉、板油(suet)、紫苏(sage)、培根、香肠、洋葱(Norfolk Plough Pudding for Plough Monday - Lavender and Lovage，菜谱)
  5. 成因：诺福克郡经济以农业为主，生产大麦、小麦；同时该地畜牧业不发达(Educational livestock project offers schools across Norfolk and Suffolk a unique hands-on experience. (rnaa.org.uk),该地农业协会官网只有一篇关于畜牧业的文章，还是一个学生参观活动)，故食物中没有鲜肉，而只有培根、香肠这样的腌制肉

伊斯兰地区——诺鲁孜节 #

Nowruz - Wikipedia

  1. 大部分伊斯兰地区都过此节
  2. 传统食物:Samanu(没错我又不知道这玩意中文名叫啥了)Samanu - Wikipedia
  3. 食材：麦芽
  4. 文化：该节日在伊斯兰文化中是新年，而这种食物是作为一种祭品而起源的，由于制法简单而流传至今
  5. 成因：中东地区生产小麦

印度——欧南节 #

Onam - Wikipedia

  1. 食物:Onamsadya(中文名……不知道……)
  2. 食材:大米、Parippu、Sambar、Rasam、Avial、Kaalan、Olan、Koottukari、Erissery等多种印度特色食材(我一个都不认识……Sadya - Wikipedia在Typical ingredients一栏有各个食材的介绍，请善用浏览器翻译功能)
  3. 文化：欧南节是印度的丰收节，主要由印度教徒庆祝，以纪念曾经统治喀拉拉邦的神话国王Daitya国王Mahabali
  4. 成因：正由于其丰收节的性质，印度人民将农业劳动的收获制成各种印度特色食材，并将其用香蕉叶盛装

俄罗斯——东正教圣诞节 #

  1. 食物：库提亚(一种在当地流行的甜品)Kutia - Wikipedia
  2. 文化:东正教是基督教的分支，也庆祝圣诞节，但由于历法不同，其圣诞节比一半认为的天主教圣诞节要迟13天。
  3. 食材：麦仁、蜂蜜等
  4. 成因：类似于中东地区，俄罗斯也盛产小麦(世界产量第三)，所以产生了这种以麦仁为原料的甜品
